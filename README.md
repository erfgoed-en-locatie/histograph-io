# Histograph IO

With Histograph IO, you can import [Newline delimited JSON](http://ndjson.org/) files into [Histograph](http://histograph.github.io).

## Redis

Histograph IO needs Redis!

Install Redis:

    brew install redis

Start Redis:

    redis-server /usr/local/etc/redis.conf
